Web Projects

Scott Vincent Design

The site you're currently on is an exercise in lightweight web design. Utilizing Google's Accelerated Mobile Pages guidelines, it ensures incredibly quick page load times. Regular web pages in this day and age are getting lazy when adding large libraries and heavy dependancies to accomplish fairly simple tasks. Bloated and inconsiderate web pages currently are the norm for users to visit.

This site loads less than 2.0MB onto your device to deliver all its information. Along with the shear speed in which it loads, it still takes into consideration the browser size and device type to deliever a responsive user friendly experience regardless of how you got here.

Media Sonar

Media Sonar is a social media monitoring tool used for public safety and law enforcement. I was a part of the project from 2013 to 2016. After working on the first version of the application briefly in 2013, I was brought on as the Lead Designer for the version 2 refresh at the end of 2014. Prototyping, designing, and styling the front-end was an indepth process with a web application that complex.

Bootstrap 3 was the base framework used to build it fully responsive to fit any screen size. Utilizing LESS css, I created a variable styling structure that allows for multiple colour themes for users to switch between.

Diply

Diply was an entertainment website that I worked on from 2013 to the end of 2014. Created initially as a online product showcase, it transformed into articles to be shared on social media.

Built with Bootstrap 3, and leveraging LESS css. Diply is an Angular application with a custom content management system. I was the Lead UI Designer during the site's first year, responsible for prototyping, front-end styling, and user experience. From launch over the next year those pages were viewed over 1 billion times in that period.
